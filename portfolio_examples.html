OpenGL Particle System - GitHub

Classwork | January - April 2024

This project was created for my advanced graphics course taken in my last semester at UPEI. I used linked list memory management to create a pool of particles that would be killed and added as needed. The system is data-driven and able to be expanded using no additional code other than xml. I demonstrated the system's capabilities by creating two effects, one with two emitters, smoke and fire, and the other with one fog emitter. This was coded using c++ and openGL on top of a basic engine called wolf. All particle, emitter, rendering, and shader code written by me.
