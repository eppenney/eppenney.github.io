►OpenGL Particle System - GitHub

Classwork | January - April 2024

This project was developed as part of my advanced graphics course during my final semester at UPEI.

I implemented a particle system using linked list memory management to create a dynamic pool of particles, enabling efficient addition and removal as needed. The system was designed to be data-driven, allowing for easy expansion through XML without requiring additional code modifications.

To demonstrate the system's versatility, I created two distinct visual effects: one featuring two emitters for smoke and fire, and another using a single emitter to generate fog.

The entire project was built using C++ and OpenGL on top of a foundational engine called Wolf, with all particle, emitter, rendering, and shader code developed exclusively by me.

►Robertson Library Perpetual Access Rights Tracker - GitHub

Desktop Application | October 2023 – April 2024

During my time working on the Robertson Library Perpetual Access Rights Tracker, my team and I developed a comprehensive technical document over the course of four months. This document included detailed requirements, roadmaps, diagrams, and thorough planning, which laid the groundwork for the application's creation during the winter semester.

My primary focus in this project was on implementing file upload and export functionality, along with developing multithreaded loading processes to enhance performance. I also contributed significantly to the UI design, ensuring a cohesive user experience by integrating the various components developed by other team members into a unified application.
